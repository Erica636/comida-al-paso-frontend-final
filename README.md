# Comida al Paso - Frontend

Frontend desarrollado con React para gestionar productos de un negocio gastronómico.

## Tecnologías
- React 18
- Vite
- React Router
- Tailwind CSS

## Instalación
```bash
npm install
npm run dev
```

La aplicación estará en `http://localhost:5173`

## Características
- Catálogo de productos y categorías
- Sistema de autenticación
- Panel de administración
- Integración con API REST (Django)
